\# Jealausy2Channel



Raspberry Pi Node‑RED UI for the Jealausy2Channel sunshade control system  

Controllino Mini firmware for two‑channel shading automation  

Includes Modbus-based communication for weather sensors and system integration



visit my website: https://colmuspro.eu/  

contact: info@colmuspro.eu



\## Project Status

Jealausy2Channel is the current stable version of the sunshade control system.  

It is feature‑complete, uses Modbus for sensor integration, and is actively used in production environments.  

No further major changes are planned at the moment.
